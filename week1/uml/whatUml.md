# What is UML (Unified Modeling Language)

## UML (Unified Modeling Language)

### Unified

Combines the most `important OO concepts`

### Modeling

`Visual` representation of Complex `Systems`.

### Language

Provides `syntax` to `define` these Models.

## UML `Modeling Language`

### Describe Application `Requirements`

- Functionality
- looks
- speed
- reliability

### Model

- different views
- different detail grades/levels

### language

- notations (Symbols)
- Rules
  - Syntax
  - Semantics

- not exact
  - different formulations
  - UML knowledge != good model
